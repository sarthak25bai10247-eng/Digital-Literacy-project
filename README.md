 OVERVIEW

This repository showcases my Digital Literacy Project for the course CSE0001 at VIT Bhopal.

 The project focuses on building essential digital skills required in today’s modern, technology-driven world. It highlights:

Awareness of digital tools
Professional online presence
Effective communication practices
Cybersecurity and safe internet usage

 The project is organized into five structured modules, each supported with relevant artifacts like screenshots, documents, and reflections, along with a detailed final report.

 PROJECT MODULES
 Task 1 – Digital Literacy Infographic

Designing an informative infographic explaining key digital literacy concepts.

 Task 2 – Student Digital Portfolio

Creating a personal portfolio to showcase academic achievements, skills, and projects.

 Task 3 – Coding & Collaboration Platforms

Exploring platforms like GitHub and understanding collaboration in software development.

 Task 4 – Professional Communication & Email Etiquette

Learning formal email writing, communication skills, and responsible social media usage.

 Task 5 – Cybersecurity & Cybercrime Case Study

Analyzing real-world cybercrime cases and learning preventive measures for safe digital practices.
